# College Score Card Crawler

Work in progress <br/>

## About 
Crawls the college score card website and puts all the data into a mongo collection  

## Requirements
Node <br/>
MongoDB <br/>
API Key from https://api.data.gov/signup/ 

##Get Started
clone repo <br/>
go to root dir <br/>
Edit crawl.js and add your API Key <br/>
node crawl.js <br/>
